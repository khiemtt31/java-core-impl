# Set

This folder contains runnable Java examples and interview-style notes for section 6 in the root `README.md`.

## What is included

- `SetApp.java` runs every demo in one pass.
- Each topic also has its own runnable class so you can execute one concept at a time.
- Each interview question has a matching `.md` file with an explanation and code-backed answer.

## How to run

From the repository root:

```bash
javac set/*.java
java -cp set SetApp
```

To run a single topic:

```bash
java -cp set HashSetDemo
```

## Learning order

1. `HashSet`
2. `LinkedHashSet`
3. `TreeSet`
