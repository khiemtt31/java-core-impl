# List

This folder contains runnable Java examples and interview-style notes for section 5 in the root `README.md`.

## What is included

- `ListApp.java` runs every demo in one pass.
- Each topic also has its own runnable class so you can execute one concept at a time.
- Each interview question has a matching `.md` file with an explanation and code-backed answer.

## How to run

From the repository root:

```bash
javac list/*.java
java -cp list ListApp
```

To run a single topic:

```bash
java -cp list ArrayListDemo
```

## Learning order

1. `ArrayList`
2. `LinkedList`
3. `Vector` (basic knowledge)
