# OOP Concepts

This folder contains runnable Java examples and interview-style notes for section 3 in the root `README.md`.

## What is included

- `OopConceptsApp.java` runs every demo in one pass.
- Each topic also has its own runnable class so you can execute one concept at a time.
- Each interview question has a matching `.md` file with an explanation and code-backed answer.

## How to run

From the repository root:

```bash
javac oop-concepts/*.java
java -cp oop-concepts OopConceptsApp
```

To run a single topic:

```bash
java -cp oop-concepts EncapsulationDemo
```
