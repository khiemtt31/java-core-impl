# Inheritance

**Question**

What is inheritance? -> Let one class reuse another class's code and add its own features.

**Answer**

Inheritance lets one class reuse and specialize another class.
It is an `is-a` relationship: `Dog` is an `Animal`.

In `InheritanceDemo.java`, `Dog` inherits shared behavior like `eat()` and `sleep()` from `Animal`.
It also adds its own `sound()` method.

**Why it matters**

- reduces duplication
- models shared behavior
- makes specialization clear
